# visualize-ai-tools
*A sample report in Power BI using a dataset of AI tools*

## Dataset
The dataset used for this sample report is The "5000 AI Tools Dataset", retrieved from Kaggle [here](https://www.kaggle.com/datasets/muhammadtalhaawan/ai-5000-tools-2023/). 

## Reading the report
The `.pbix` file is configured to preprocessed files in Google Drive, so you should only need to download the file and open it locally. You don't need to run the preprocessing notebook.
